# Smart-Bus-Ticketing-System
An automated bus ticketing platform featuring QR-based check-in/out, GPS distance-based fare calculation, and an integrated in-app wallet for seamless, cashless payment. Designed to enhance passenger convenience and streamline bus operations with real-time ticketing and fare management.
